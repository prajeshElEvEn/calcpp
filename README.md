# calcpp

a calculator cpp package built with c++

## installation

- fork and clone the repo

```bash
git clone <repo_url>
```

- navigate to the directory

```bash
cd calcpp
```

## build

```bash
mkdir build
cd build
cmake ..
cmake --build .
```

## usage

- enter build directory

```bash
cd build
```

- run `./calculator`

```bash
./calculator
```
